clausewitz_parser: A parser for Clausewitz (Paradox) files
==========================================================

## Overview

## Requirements

- Rust nightly (2018-05-10)

## Usage

Put this in your Cargo.toml:

```toml
[dependencies]
clausewitz_parser = "0.1"
```

Then put this in your crate root:

```rust
extern crate clausewitz_parser;
```
